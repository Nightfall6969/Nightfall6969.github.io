TRUE - The motherboard is the main circuit board. It provides physical connections and communication pathways between the CPU, RAM, storage devices, expansion cards, and peripherals. Every component connects to the motherboard either directly or through cables.

üî≤ Key Motherboard Components

  • CPU Socket: Houses the processor
  • RAM Slots: Hold memory modules
  • Chipset: Controls data flow
  • PCIe Slots: For graphics cards, SSDs
  • SATA Ports: Connect HDDs, SSDs
  • USB Headers: Front panel ports

üîå I/O & Power

  • ATX Power: 24-pin main power
  • CPU Power: 4/8-pin power
  • Audio Jacks: Sound I/O
  • Ethernet Port: Network
  • USB Ports: USB 2.0, 3.0, 3.1, USB-C
  • BIOS Chip: Firmware storage

üîÑ Communication Pathways: The motherboard contains buses (data pathways) like the Front-Side Bus (older), Direct Media Interface (DMI), and PCIe lanes. These transfer data between components at speeds up to 64 GB/s for PCIe 5.0!
üí° Tech Tip: When building a PC, ensure your case supports your motherboard form factor. Also check CPU socket compatibility - an Intel CPU won't fit in an AMD socket! The chipset (Z790, B650, etc.) determines features like overclocking support and how many USB/PCIe lanes are available.
